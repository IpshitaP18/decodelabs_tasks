## 🤖 Project 1 — Nova (Rule-Based AI Chatbot)

![Python](https://img.shields.io/badge/AI-Rule%20Based-purple.svg)
![Streamlit](https://img.shields.io/badge/UI-Streamlit-red.svg)

A rule-based chatbot that responds using keyword matching and predefined logic (no ML or APIs).

📌 Features:

* Keyword-based conversation engine
* Time/date responses
* Joke generator
* Session-based chat memory
* Sidebar quick prompts

📁 Folder: `Project-1/`
📖 Details: See project README

## 🎓 Project 2 — Student Pass/Fail Prediction

![ML](https://img.shields.io/badge/ML-Decision%20Tree-green.svg)
![Scikit-Learn](https://img.shields.io/badge/Library-ScikitLearn-orange.svg)

A machine learning web app that predicts student performance using a Decision Tree model.

📌 Features:

* Pass/Fail classification
* Data visualization dashboard
* Model accuracy display
* Encoded categorical features

📁 Folder: `Project-2/`
📖 Details: See project README

## 🎵 Project 3 — Spotify AI Recommender

![Recommendation](https://img.shields.io/badge/AI-Recommendation-blueviolet.svg)

A content-based music recommender system using cosine similarity.

📌 Features:

* Mood-based filtering
